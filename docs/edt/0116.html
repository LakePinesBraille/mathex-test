Moving to Line Start/End

Typing [Home] or [End] moves the input position back or forward to the start or end of each line.

Moving backward moves to the left of the first character in each line. Moving forward moves to the right of the last character in each line.

To try these out, use [Home] and [End] multiple times and observe how the input position changes.

Press [Alt+Enter] when you are ready to continue.
