You have completed the topic Using the Tutorial, which covers how to move around in the tutorial.

[Alt+Enter]

Move to the next page

[Alt+Left]

Move backward to the previous page

[Alt+Right]

Move forward to the next page

[Alt+Up]

Move to the previous document link

[Alt+Down]

Move to the next document link

[Alt+Shift+X]

Open or close the editor panels

[Alt+Shift+T]

Open the tutorial

[Esc]

Close the tutorial

[Alt+Home]

Move to the first page of the current topic

[Alt+End]

Move to the last page of the current topic

[Alt+Shift+I]

Move to the tutorial index

Press [Alt+Enter] when you are ready to continue to the next topic.
