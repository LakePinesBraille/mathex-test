Getting To Know the Equalize Editor

[Drive] Save - How do I save an existing shared document?

Store a document to the shared document last used to access it.

This menu operation stores the document using HTML and Content MathML to allow the Open operation to retrieve the text and math portions of the document.

You will not be prompted to supply the name of the document, unless it has not been saved before.

Type Alt Shift D, S to store a document to the shared document last used to access it.

This operation modifies the shared document in the shared file system.
