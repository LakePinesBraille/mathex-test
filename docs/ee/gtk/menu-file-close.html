Getting To Know the Equalize Editor

[File] Close - How do I close the current document?

Clear the contents of the current document.

This menu operation discards the contents of the current document, so you can enter the text and math contents for a new document.

Once you have created the contents of the new document, use the Save As operation to store the document to the file system.

Type Alt Shift F, C to clear the contents of the current document.

This operation does not modify any file in the file system.
