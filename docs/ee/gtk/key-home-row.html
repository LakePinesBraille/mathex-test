Home Row Braille - How do I enter braille on a computer keyboard?

Home Row input mode simulates a refreshable braille keyboard using the home row keys of a standard computer keyboard to generate braille symbol input.

Home Row input mode may be used with a computer keyboard by sighted users to simulate braille input, or by braille users who do not have a refreshable braille device.

To enter braille in Home Row input mode, use the home row keys asdf jkl; to enter braille dots-7321 4568 respectively.

Control characters may be entered while in Home Row input mode using the Ctrl key. Menus and menu items may be activated using the Alt key.

While sighted users are likely to prefer Computer Keyboard input mode, and braille users are likely to prefer Refreshable Braille input mode, Home Row input mode can prove useful.

Type Ctrl h or ==h or dots-123456 dots-123456 dots-125 to switch to Home Row input mode.

The current keyboard input mode may be changed on the Settings Page, and persists when you revisit the EE web application.
