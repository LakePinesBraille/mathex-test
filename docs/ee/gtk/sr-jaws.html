Getting To Know the Equalize Editor

JAWS Setup - How do I install the EE configuration file?

Summary

Download the file ee.jcf into your JAWS Settings Folder, and make sure the name of the file is www.lakepinesbraille.com.jcf.

Step by Step - Install the EE Configuration File for JAWS

The Equalize Editor provides a JAWS configuration file called ee.jcf which needs to be saved into your JAWS Settings Folder with the name www.lakepinesbraille.com.jcf

This JAWS configuration file ee.jcf contains configuration directives for JAWS that can be used to apply recommended settings for the EE. You can use the following steps to install these settings as a Domain Settings file in the JAWS Settings Folder, so they will be used for the EE web site without affecting configuration settings for other applications.

Using this name instructs JAWS to use this configuration, but only when using a browser that is viewing the web site www.lakepinesbraille.com.

This way the configuration needed for the Equalize Editor does not affect your using JAWS with other applications, or with other web sites.

Use the link below to download the JAWS configuration file for the Equalize Editor.

Save this file in your JAWS Settings Folder on your local machine:

C:\Users\[username]\AppData\Roaming\Freedom Scientific\JAWS\[year]\Settings\enu

where [username] is your Windows user name, and [year] is your JAWS version year.

www.lakepinesbraille.com.jcf

When saving the file, make sure to put it in the JAWS Settings Folder, with the file name as shown above.
