Lesson 11: Braille Keyboards

This lesson shows how to use Computer Braille input from an electronic braille keyboard.

So far we have used Computer Braille Input mode to simulate the braille symbols used by an electronic braille keyboard.

Now we will use Computer Braille Input to enter these braille symbols directly from an electronic braille keyboard.

Example 1: Switch to Computer Braille Input.

Just as before, you type Ctrl‑r to switch to Computer Braille Input mode, and Ctrl‑q to return to Qwerty Input mode.

Example 2: Use your braille keyboard.

On your electronic braille keyboard, use the braille keys to enter the braille symbols directly.

If your screen reader is properly set up expect Computer Braille code from your braille keyboard, and the editor is properly in Computer Braille Input mode, you do not need to know the Computer Braille codes, just the braille symbols.

When entering text, type the proper English braille contracted symbols. When typing math, enter the proper Nemeth braille math symbols. In text mode:

Type ⠬ for ing.

Type ⠤ for hyphen.

Type ⠡ for ch.

Type ⠌ for st.

Type ⠘ for superscript.

Type ⠐ for baseline.

Type ⠠ for capital.

There are many more. The complete list is defined by The Rules of Unified English Braille and The Nemeth Braille Code for Math and Science Notation.

Example 3: Enter braille contractions.

Any braille contractions entered by the user are expanded into literary text at the end of each word.

So typing the letters

b y c n q h x j z p w l

in Computer Braille Input mode will produce the text:

but you can not quite have it just as people will like

All of the UEB contractions for literary text, as well as the Nemeth Braille symbols for math, may be used in from your braille keyboard.
