Lesson 7: Embossing Braille

This lesson shows how to produce hard copy braille from documents you create.

Example 1: View a BRF document.

The editor produces contracted literary braille text with embedded Nemeth braille math. It displays this braille in the panel below the input area, and on an attached electronic braille device with a screen reader.

The editor is also able to export ASCII braille as a BRF file. When you use View BRF on the File menu, the BRF encoding for the current document is displayed in a separate browser tab.

Dismiss the tab when you are ready to return to the editor window.

Example 2: Save a BRF document.

When you use the Save BRF item on the File menu, the BRF encoding is saved in a local file that you can choose using the standard save file dialog.

Example 3: Emboss a BRF document.

When you use the Emboss BRF item on the File menu, the BRF encoding for the current document is displayed in a separate browser tab, and then the browser Print operation is invoked to allow you to emboss the document to a braille embosser.

Dismiss the tab when you are ready to return to the editor window.

Exercises

1. View the current document as a BRF document using View BRF.

2. Save the current document as a BRF document using Save BRF.

3. Emboss the current document to hard copy braille using Emboss BRF.
