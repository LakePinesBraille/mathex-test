Lesson 9: Home Row Input

This lesson shows how to use the home row keys.

So far we have used Qwerty Input mode to enter text and math. It is designed for sighted users who are using a computer keyboard.

Now we will use Home Row Input mode to enter braille symbols. It is designed for braille users who are not using a braille keyboard.

Example 1: Switch to Home Row Input.

Type Ctrl‑h to switch to Home Row Input mode, and Ctrl‑q to return to Qwerty Input mode.

The editor reports the new input mode using a visual popup dialog, announced by the screen reader.

Example 2: Locate the home row keys.

In Home Row Input mode, the keyboard keys A S D F and J K L ; are used to enter dots 7 3 2 1 and 4 5 6 8 respectively.

The dot positions are arranged just like those on a braille keyboard.

The editor listens for braille symbols typed as multiple keys, and uses them to create text and math just like single keys are used in Qwerty Input mode.

Example 3: Enter braille contractions.

Example 4: Nemeth switch indicators.

When typing inline math formulas, you use the Nemeth open (⠸⠩) and close (⠸⠱) switch indicators to begin and end math mode.

To create display math formulas, use an extra dot‑456 (⠸⠸⠩ and ⠸⠸⠱) with these indicators to open and close a display math formula.

Hint: Remember to make the braille panel visible when using Home Row Input mode, as it can remind you of braille symbols you would like to type.
