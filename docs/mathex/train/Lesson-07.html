Lesson 7: Previewing Braille

This lesson shows how to view online braille.

Example 1: View a BRL document.

The editor produces contracted braille text with Nemeth braille math. It displays the braille in a panel below the input area, and on an attached electronic braille device with the help of a screen reader.

The editor can also export Unicode braille as a BRL file. When you use View BRL on the File menu, the BRL encoding for the document displays in a separate browser tab.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Example 2: Save a BRL document.

When you use Save BRL on the File menu, the BRL encoding is saved in a local file that you choose using the standard save file dialog.

You can also use Save BRL on the Drive menu to save the BRL encoding to a file on Google Drive.

Example 3: Print a BRL document.

When you use Print BRL on the File menu, the BRL encoding for the document displays in a separate browser tab, and then the browser print dialog is invoked to allow you to print the document to an attached printer.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Exercises

1. View the current document as a BRL document using View BRL.

2. Save the current document as a BRL document using Save BRL.

3. Print the current document to hard copy Unicode braille using Print BRL.
