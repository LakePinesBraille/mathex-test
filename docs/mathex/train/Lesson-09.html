Lesson 9: Braille Devices

This lesson shows how to use computer braille input from a computer keyboard or from an electronic braille keyboard.

So far we have used Qwerty Input mode and Home Row Input mode to enter text and math.

Now we will use Braille Input mode to enter braille symbols. It is designed for braille users with an electronic braille keyboard.

It may also be used to enter computer braille on a computer keyboard to simulate input from an electronic braille keyboard.

Example 1: Switch to Braille Input.

You type Ctrl‑r to switch to Braille Input mode, and Ctrl‑q to return to Qwerty Input mode.

Example 2: Type computer braille.

In computer braille, letters and numbers represent themselves. Other special keys are used to enter braille symbols that do not represent letters or numbers.

These symbols are the ones used by electronic braille keyboards to send braille to the screen reader, but they may also be typed directly on a computer keyboard.

So typing the letters

b y c n q h x j z p w l

in Braille Input mode will produce the text:

but you can not quite have it just as people will like

All of the UEB contractions for literary text, as well as the Nemeth Braille symbols for math, may be used in Braille Input mode.

Example 3: Use a braille keyboard.

The braille keys provided by most braille keyboards may be used to enter braille symbols directly without having to remember the Computer Braille codes. It only requires the screen reader be configured to expect Computer Braille Input from the braille keyboard.
