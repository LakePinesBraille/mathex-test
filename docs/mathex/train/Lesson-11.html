Lesson 11: Screen Readers

This lesson shows how to set up your screen reader.

Each screen reader allows you to choose the braille code used to translate input from an electronic braille keyboard.

The Braille Input code selection typically includes choosing an input language, choosing a braille standard, and choosing Computer Braille, Uncontracted Braille, or Contracted Braille as the braille encoding.

Since the editor performs its own braille translation, it needs to receive braille symbols from the braille keyboard without having them translated by the screen reader.

The recommended screen reader setting is to use Computer Braille as the input translation table. Doing so allows the editor to receive the raw braille symbols used to create text and math.

Example 1: JAWS setup.

To select Computer Braille input and other useful settings when using JAWS, follow these steps.

1. Type JAWS + 6 to open the JAWS Settings Center.

2. Under Braille > General > Translation:

2.1 Set Input to Computer Braille.

2.2 Set Output to UEB Grade 2.

2.3 Set Show current word in Computer Braille to Off.

3. Under Forms Mode:

3.1 Set Select Forms Mode to Auto.

4. Under Web/HTML/PDFs > Miscellaneous:

4.1 Set Page Refresh Filter to Suppress All Refreshes.

Alternatively, you can download the file

ee/config/ee.jcf and save it with the name www.lakepinesbraille.com.jcf in the JAWS Settings Folder.

C:\Users\LPB\AppData\Roaming\Freedom Scientific\JAWS\2024\Settings\enu

[Replace LPB with your Windows username.]

The editor provides this JAWS configuration file to change these settings for the editor web page, without changing settings for other applications.

See the editor documentation for more details.

Example 2: NVDA setup.

To select Computer Braille input and other useful settings when using NVDA, follow these steps.

1. Type NVDA + N to open the NVDA menu.

2. Under Preferences > Settings > Braille:

2.1 Set Input Table to Unicode Braille.

2.2 Set Output Table to UEB Code Grade 2.

2.3 Set Expand to computer braille for the word at the cursor to Off.

2.4 Set Show Messages to Disabled.

3. Under Browse Mode:

3.1 Set Automatic focus mode for focus changes to On.

3.2 Set Automatic focus mode for caret movement to On.

3.3 Set Automatically set system focus to focusable elements to On.

Alternatively, you can download the file

ee/config/ee.ini and save it in the User Configuration Folder.

C:\Users\LPB\AppData\Roaming\nvda

[Replace LPB with your Windows username.]

The editor provides the EE gestures file ee/config/gestures.ini that may be added to the NVDA gestures file gestures.ini to assign the keyboard gesture NVDA + E to activate and deactivate the ee configuration profile, to change these settings for the editor, without changing settings for other applications.

See the editor documentation for more details.

Exercises

1. Set up your screen reader to connect to your electronic braille keyboard.

2. Set up your screen reader to expect Computer Braille input from your electronic braille keyboard.

3. Confirm that braille output produced by the editor is shown on your electronic braille display.

4. Confirm that braille input from your electronic braille keyboard produces input in the editor input area.

Press [Alt+Enter] when you are ready to continue.
