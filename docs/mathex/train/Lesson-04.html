Lesson 4: Entering Math

This lesson shows how to create math formulas.

To enter a math formula, place the input cursor where you would like the math to appear and type Ctrl‑m to enter math mode.

When you enter math mode, the editor creates an empty box for you to fill in.

In math mode, special keys are used to create math operations. Some are easy to understand, others are less obvious.

Type + for addition.

Type ‑ for subtraction.

Type * for multiplication.

Type / for vertical fractions.

Type Ctrl‑/ for inline fractions.

Type ^ for exponentiation.

Type ` for square roots.

Type Ctrl‑` for general roots.

Type = for an equation.

Type < for less than.

Type > for greater than.

Type Ctrl‑< for less than or equal to.

Type Ctrl‑> for greater than or equal to.

There are many more, but these are enough to create many common formulas.

Hint: after entering an exponent, you can use ) or Right arrow to indicate that you are finished with the exponent, and want to return to the baseline.

Example 1: Enter the quadratic equation.

The general quadratic equation is easier than you might expect.

a x 2 b x c 0

To enter this equation, you type the symbols ax^2)+bx+c=0 in math mode.

Example 2: Enter the general solution.

The general formula for the solutions to the quadratic equation is the one you may have learned once, and promptly forgot!

x b b 2 4 a c 2 a

This equation is a bit more involved, of course. You can create it by typing x=/‑b+‑`b^2)‑4ac[tab]2a

in math mode, although there are other ways that work just as well.

Hint: the tab key moves to the next empty box.

Each of these math operations (and many, many more) can also be found on buttons along the top of the input area, and on the expandable palettes on the right.

Exercises

Create a copy of each of the following formulas.

1. a x b 0

2. y m x b

3. A r 2

(Hint: the greek letter pi can be typed as pi in math mode.)

4. x h 2 a 2 y k 2 b 2 r 2

Homework

Create a copy of each of the following formulas.

1. m y 1 y 0 x 1 x 0

(Hint: a numeric subscript may be typed as the letter followed by the numeral, followed by right arrow to return to the baseline.)

2. x 2 x 2 1

(Hint: you can enter sin and cos by typing the names of the functions)

3. x h 2 a 2 y k 2 b 2 1

4. D x 1 x 0 2 y 1 y 0 2

Press [Alt+Enter] when you are ready to continue.
