Lesson 10: Computer Braille

This lesson shows how to use Computer Braille input.

So far we have used Qwerty Input mode and Home Row Input mode to enter text and math.

Now we will use Computer Braille Input mode to enter braille symbols. It is designed for braille users who are using an electronic braille keyboard.

It can also be used to enter computer braille on a computer keyboard to simulate input from an electronic braille keyboard if one is not available.

This lesson shows how to use Computer Braille Input mode on a computer keyboard. The next lesson shows how to set up your Screen Reader to expect Computer Braille input, and the lesson after that shows how to use an electronic Braille Keyboard if you have one available.

Example 1: Switch to Computer Braille Input.

Type Ctrl‑r to switch to Computer Braille Input mode, and Ctrl‑q to return to Qwerty Input mode.

The editor reports the new input mode using a visual popup dialog, also announced by the screen reader.

Example 2: Type Computer Braille input.

In Computer Braille Input mode, you type letters as themselves, and use numbers and other special keys to enter braille symbols that do not represent letters.

These symbols are the same ones used by electronic braille keyboards to send braille to the screen reader, but they may also be typed directly on a computer keyboard.

Some computer braille codes are the same as the Nemeth braille for the symbol, while others are not as easy to remember.

Type + for ⠬ (ing).

Type ‑ for ⠤ (hyphen).

Type * for ⠡ (ch).

Type / for ⠌ (st).

Type _ for ⠸ (modifier).

Type % for ⠩ (sh).

Type : for ⠱ (wh).

Type ~ for ⠘ (superscript).

Type ; for ⠰ (english letter).

Type " for ⠐ (baseline).

Type , for ⠠ (capital).

There are many more. The complete list is defined by the Computer Braille Code standard.

Example 3: Enter braille contractions.

Any braille contractions entered by the user are expanded into literary text at the end of each word.

So typing the letters

b y c n q h x j z p w l

in Computer Braille Input mode will produce the text:

but you can not quite have it just as people will like

All of the UEB contractions for literary text, as well as the Nemeth Braille symbols for math, may be used in Computer Braille Input mode.

Example 4: Nemeth switch indicators.

When typing inline math formulas, you use the Nemeth open (_%) and close (_:) switch indicators to begin and end math mode.

To create display math formulas, use an extra dot‑456 (__% and __:) with these indicators to open and close a display math formula.
