Lesson 12: Braille Keyboards

This lesson shows how to use electronic braille input.

So far we have used Computer Braille Input mode to enter braille symbols.

Now we will use Computer Braille Input mode to enter braille symbols using an electronic braille keyboard.

Example 1: Switch to Computer Braille Input.

Type ⠿⠿⠗ for Ctrl‑r to switch to Computer Braille Input mode, and ⠿⠿⠟ for Ctrl‑q to return to Qwerty Input mode.

The editor reports the new input mode using a visual popup dialog, also announced by the screen reader.

Example 2: Use your braille keyboard.

On your electronic braille keyboard, use the braille keys to enter braille symbols directly.

You do not need to know the Computer Braille codes, just the braille symbols.

When entering text, type the proper English braille contracted symbols. When typing math, enter the proper Nemeth braille math symbols. In text mode:

Type ⠬ for ing.

Type ⠤ for hyphen.

Type ⠡ for ch.

Type ⠌ for st.

Type ⠸ for modifier.

Type ⠩ for sh.

Type ⠱ for wh.

Type ⠘ for superscript.

Type ⠰ english letter indicator.

Type ⠐ for baseline indicator.

Type ⠠ for capital letter indicator.

There are many more. The complete list is defined by The Rules of Unified English Braille and The Nemeth Braille Code for Math and Science Notation.

Example 3: Enter braille contractions.

Any braille contractions entered by the user are expanded into literary text at the end of each word.

So typing the braille symbols for the letters

⠃⠀⠽⠀⠉⠀⠝⠀⠟⠀⠓⠀⠭⠀⠚⠀⠵⠀⠏⠀⠺⠀⠇

in Computer Braille Input mode will produce the text:

but you can not quite have it just as people will like

All of the UEB contractions for literary text, as well as the Nemeth Braille symbols for math, may be used from your braille keyboard.

Example 4: Nemeth switch indicators.

When typing inline math formulas, you use the Nemeth open (⠸⠩) and close (⠸⠱) switch indicators to begin and end math mode.

To create display math formulas, use an extra dot‑456 (⠸⠸⠩ and ⠸⠸⠱) with these indicators to open and close a display math formula.
