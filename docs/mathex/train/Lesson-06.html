Lesson 6: Printing a Document

This lesson shows how to print your documents.

The end of this document includes the complete derivation of the solution of the quadratic equation.

In principle, you should now be able to create this document from scratch, with a bit of effort.

Let's see what you can do with these documents.

Example 1: Preview an HTML document.

The editor creates HTML documents that include Content MathML, which web browsers do not support.

However, the editor can export HTML documents that use Presentation MathML, which most web browsers now do support.

When you select Print Preview on the File menu, the document displays in a separate browser tab where you can read it like any other web page.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Example 2: Save an HTML document.

When you select Save HTML on the File menu, the HTML encoding is saved in a local file that you choose using the standard save file dialog.

You can also use Save HTML on the Drive menu to save the HTML encoding to a file on Google Drive.

The editor adds the letters -p to the end of the HTML file name to remind you it is only a preview of the file.

Example 3: Print an HTML document.

When you select Print HTML on the File menu, the document displays in a separate browser tab, and the browser print dialog is used to print the document to a printer, or save it as a PDF file, just like with any other web page.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Example 4: View HTML source markup.

You can view the HTML and Content MathML markup used to represent a document using View Source on the File menu.

The HTML source markup displays in a separate browser tab. Press Esc to dismiss this tab when you are ready to return to the editor window.

This HTML markup is what the editor saves when it saves a document.

Example 5: View HTML output markup.

You can view the HTML and Presentation MathML markup used to print a document using View HTML on the File menu.

The HTML output markup displays in a separate browser tab. Press Esc to dismiss this tab when you are ready to return to the editor window.

This HTML markup is what the editor uses when it previews or prints a document.

Exercises

1. Preview the current document as an HTML document using Print Preview.

2. Save the current document as an HTML document using Save HTML.

3. Print the current document as an HTML document using Print HTML.

4. View the source markup for the current document using View Source.

5. View the output markup for the current document using View HTML.
