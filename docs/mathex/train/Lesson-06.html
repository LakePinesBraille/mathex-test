Lesson 6: Printing a Document

This lesson shows how to produce hard copy from the documents you create.

The end of this document includes the complete derivation of the formula for the general solution of the quadratic equation.

You should now be able to create this document from scratch, with a bit of work.

Now we will show what you can do with these documents.

Example 1: Export an HTML document.

The documents created by the editor use a special kind of HTML that includes Content MathML, which web browsers do not support.

However, the editor can export a version of HTML that uses Presentation MathML, which many web browsers do support.

When you select the Export HTML item on the File menu, the current document is displayed in a static browser tab where you can read it like any other web page.

Dismiss the tab when you are ready to return to the editor window.

Example 2: Print an HTML document.

When you select the Print HTML item on the File menu, the current document is displayed in a separate browser tab, and the browser Print operation is used to print the document to a printer or save it as a PDF file, just like with any other web page.

Dismiss the tab when you are ready to return to the editor window.

Example 3: View HTML source markup.

You can view the HTML and Content MathML markup used to represent a document using the View Source item on the File menu.

The HTML source markup is displayed in a separate browser tab. Dismiss the tab when you are ready to return to the editor window.

This HTML markup is what the editor saves when it saves a document.

Example 4: View HTML output markup.

You can view the HTML and Presentation MathML markup used to print a document using the View HTML item on the File menu.

The HTML output markup is displayed in a separate browser tab. Dismiss the tab when you are ready to return to the editor window.

This HTML markup is what the editor uses when it exports or prints a document.

Exercises

1. Export the current document as an HTML document using Export HTML.

2. Print the current document as an HTML document using Print HTML.

3. View the source markup for the current document using View Source.

4. View the output markup for the current document using View HTML.
