Lesson 1: Using Local Files

This lesson shows how to save your work.

Example 1: Save a copy of this file.

When you select Save As from the File menu, the document is saved in a local file that you choose using the standard file save dialog.

When you save a copy of a file, the original file is unchanged.

Example 2: Modify your copy of this file.

These training lessons are ordinary editor documents just like any other. Once you have saved your local version, you can modify the document as much as you like.

Example 3: Save the new version.

After modifying the document, you can use the Save item on the File menu to make sure your local copy has your latest changes.

Exercises

1. Make a local copy of this file.

(using File > Save As)

2. Modify the document to make it your own.

3. Save your updated version.

(using File > Save)

4. Close the document.

(using File > Close)

5. Open your updated version.

(using File > Open)

Homework

1. Create a new empty file.

(using File > New)

2. Modify the file to include a description of a document you would like to create.

3. Save your modified version.

(using File > Save)
