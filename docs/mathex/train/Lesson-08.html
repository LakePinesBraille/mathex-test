Lesson 8: Embossing Braille

This lesson shows how to produce hard copy braille.

Example 1: View a BRF document.

The editor produces contracted braille text with Nemeth braille math. It displays the braille in a panel below the input area, and on an attached electronic braille device with the help of a screen reader.

The editor can also export ASCII braille as a BRF file. When you use View BRF on the File menu, the BRF encoding for the document displays in a separate browser tab.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Example 2: Save a BRF document.

When you use Save BRF on the File menu, the BRF encoding is saved in a local file that you choose using the standard save file dialog.

You can also use Save BRF on the Drive menu to save the BRF encoding to a file on Google Drive.

Example 3: Emboss a BRF document.

When you use Emboss BRF on the File menu, the BRF encoding displays in a separate browser tab, and then the browser print dialog is invoked to allow you to emboss the document to a braille embosser.

Press Esc to dismiss this tab when you are ready to return to the editor window.

Exercises

1. View the current document as a BRF document using View BRF.

2. Save the current document as a BRF document using Save BRF.

3. Emboss the current document to hard copy braille using Emboss BRF.
