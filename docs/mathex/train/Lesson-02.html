Lesson 2: Using Google Drive

This lesson shows how to save your work to Google Drive.

Example 1: Save a copy of this file.

When you select Save As from the Drive menu, the document is saved in a shared Google Drive folder that you choose using the Google Drive file save dialog.

When you save a copy of a file, the original file is unchanged.

The Google Drive file save dialog uses several steps:

1. Choose a Google Account, if you use more than one account on this machine.

2. Grant the editor permission to access your Google Drive files.

3. Select a folder to Save into. Single click the folder you want, and press Select.

4. Choose a name for the new file. Type the name into the edit box, and press OK.

Example 2: Modify your copy of this file.

You can modify files from Google Drive just as you would any other files. Once you have saved this file to Google Drive, you can modify it as much as you like.

Example 3: Save the new version.

After modifying the document, you can use the Save item on the Drive menu to make sure your copy in Google Drive has your latest changes.
