Getting To Know the Accessible Equation Editor

Initial Setup - How do I get the AEE to work?

The AEE is intended to work with various keyboard input devices and screen readers with as little special setup as possible, to meet your individual needs.

You do not need to use a screen reader to enter braille or to have braille generated for you. Visit the Home Row Braille topic to learn how. A screen reader is needed to collect braille input and to produce braille output using a refreshable braille keyboard if you have one.

Use the following links to learn how to enter print and braille symbols from your preferred type of keyboard, and how to set up your preferred screen reader to produce braille input for the AEE.
