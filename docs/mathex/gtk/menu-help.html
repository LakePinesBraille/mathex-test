Getting To Know the Equalize Editor

Help Menu - How do I get more help?

The Help Menu provides access to online documentation, configuration settings, software licensing, and version information.

Type Alt Shift H to navigate to the Help Menu.

Use the links below to explore each item on the Help Menu.

  • Welcome - Welcome to the EE! - a one-page quick-start introduction
  • Tutorial - Getting To Know the EE - this multi-page self-guided tutorial
  • Users Guide - The EE Users Guide - the detailed reference document
  • Samples - EE Samples Folder - a Google Drive folder with sample EE files
  • Settings - EE Settings Page - the editor behavior configuration page
  • Terms of Service - What may I do with the EE?
  • Privacy Policy - What may the EE do with my information?
  • About - What version of the EE am I using?
