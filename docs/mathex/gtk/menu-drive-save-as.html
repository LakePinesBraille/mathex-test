[Drive] Save As - How do I save a document to a new shared document?

Store a document to a new document in the shared file system.

This menu operation stores the document using HTML and Content MathML to allow the Open operation to retrieve the text and math portions of the document.

You will be prompted to select a shared folder where the new document will be stored, then you will be prompted to supply the name of the new shared document.

Type Alt Shift D, A to store a document a new shared document.

This operation modifies the document in the shared file system.
