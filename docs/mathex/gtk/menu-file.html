File Menu - How do I save my work?

The File Menu provides tools to load and store your documents in the local file system, to view and store them in different file formats, and to export them into a browser web page where they may be printed.

Type Alt Shift F to navigate to the File Menu.

Use the links below to explore each item on the File Menu.

  • New - How do I create a new document?
  • Open - How do I open an existing document?
  • Save - How do I save an existing document?
  • Save As - How do I save a document to a new file?
  • Close - How do I close the current document?
