Getting To Know the Equalize Editor

[Drive] Save Replace - How do I save a document to replace an existing shared document?

Store a document to replace an existing document in the shared file system.

This menu operation stores the document using HTML and Content MathML to allow the Open operation to retrieve the text and math portions of the document.

You will be prompted to select the existing document to replace, then you will be asked to confirm that you would like to replace it.

Type Alt Shift D, R to store a document to replace an existing shared document.

This operation modifies the document in the shared file system.
