Welcome to the Accessible Equation Editor!

Welcome to the Accessible Equation Editor for Braille Text and Math documents!

Within any of the AEE sample pages, press Tab to pull the input focus to the AEE input area, then enter Braille using a braille terminal. The AEE is a web application, so you need to exit application mode to interact with the enclosing web page. NVDA users press Ctrl+NVDA+Space to do so, and JAWS users press the PC Cursor command twice.

If you are a sighted user, click in the input area, and press Ctrl+q to enter QWERTY entry mode, and enter math using your computer keyboard.
