Welcome to the Accessible Equation Editor!

Welcome to the Accessible Equation Editor (AEE), a Web Application for Braille Text and Math documents!

When you load the AEE Web Application, it pulls the input focus to the AEE input area, where you may enter text and math. At any time, you can press the Escape key to return to the input area.

If you are a braille user, press Ctrl+r to select Braille input mode (the default), and enter braille text and math into the input area using your braille terminal.

If you are a sighted user, press Ctrl+q to select QWERTY input mode, and enter text and math using your computer keyboard.

To enter braille using your computer keyboard, press Ctrl+6 to select Six-Key input mode, and enter braille using the home row keys asdf jkl; to enter braille dots-7321 4568, respectively.

Please see the AEE Users Guide for more information, including a description of the AEE keyboard input modes, the AEE application menu items, screen reader configuration information, and much more!
