Welcome to the Accessible Equation Editor!

The Accessible Equation Editor (AEE) is a Web Application for Braille Text and Math documents!

Press Escape to dismiss this Welcome Page and return to the application.

Please see the Tutorial for help getting to know the AEE.

  • The Keyboard Input page describes how to select your preferred keyboard input method.

  • The Screen Readers page describes how to set up your screen reader to use the editor.

When the editor loads, it pulls focus to its input area, where you enter text and math.

  • From anywhere else on the page, press Escape to return to the input area.

Thank you for your interest in the Accessible Equation Editor!
