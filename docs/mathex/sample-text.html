Welcome to the Accessible Equation Editor!

Braille Text Entry plus Nemeth Braille Math

Press Tab to pull focus to the AEE input area, then enter uncontracted Braille using a braille terminal. The AEE is a web application, so you need to exit application mode to interact with the surrounding web page. NVDA users press Ctrl+NVDA+Space to do so, and JAWS users press the PC Cursor command twice.

If you are a sighted user, click in the input area, press Ctrl+q to enter QWERTY entry mode, and enter math using your computer keyboard. Press Ctrl+q again to switch to Six Key entry mode, and enter braille from the QWERTY keyboard. Press Ctrl+q a third time to switch back to Braille entry mode.

Please see the Nemeth Braille Math demo page to try the Nemeth Braille Math input mode, or the Palette Button Preview to see a preview of an extended set of AEE buttons.
