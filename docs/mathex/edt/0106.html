Using the Current Topic

Typing [Alt+Home] returns to the first page of the current topic, and [Alt+End] skips to its last page.

The last page of each topic provides a summary of the current topic, before moving on to the next topic.

To try these out, use [Alt+End] to move to the topic summary, then [Alt+Home] to review the first page, then use [Alt+Left] twice to return here.

Press [Alt+Enter] when you are ready to continue.
