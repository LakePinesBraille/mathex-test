Moving by Paragraphs

Typing [Ctrl+Up] or [Ctrl+Down] moves the input position up or down one paragraph at a time.

Moving up moves to the left of the first character of each paragraph, and moving down moves to the right of the last character of each paragraph.

To try these out, use [Ctrl+Up] and [Ctrl+Down] multiple times to see how the input position changes.

Press [Alt+Enter] when you are ready to continue.
