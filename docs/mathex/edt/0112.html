Using the Page History

Typing [Alt+Left] moves backward to the previous page, and typing [Alt+Right] moves forward again, just like the browser backward and forward buttons.

— Important! — These keys move along the history of pages you visit, not the order of pages in a topic.

To try these out, type [Alt+Left] to go back one page, and then [Alt+Right] to return here.

Press [Alt+Enter] when you are ready to continue.
