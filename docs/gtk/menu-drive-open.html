Getting To Know the Accessible Equation Editor

[Drive] Open - How do I open an existing shared document?

Load a document from the shared file system.

This menu operation loads an existing document from the shared file system, which must use HTML and Content MathML to represent the text and math portions of the document.

You will be prompted to sign in to your Google account if needed, then you will be prompted to select the document to be opened.

Make changes to the text and math portions of the document, and use the Save operation to store the new version back to the shared file system.

Type Alt Shift D, O to open an existing shared document.

This operation does not modify any file in the shared file system.
