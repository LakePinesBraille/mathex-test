Help Menu - How do I get more help?

The Help Menu provides access to online documentation, configuration settings, software licensing, and version information.

Type Alt Shift H to navigate to the Help Menu.

Use the links below to explore each item on the Help Menu.

  • Welcome - Welcome to the AEE! - a one-page quick-start introduction
  • Tutorial - Getting To Know the AEE - this multi-page self-guided tutorial
  • Users Guide - The AEE Users Guide - the detailed reference document
  • Settings - AEE Settings Page - the editor behavior configuration page
  • Terms of Service - What may I do with the AEE?
  • Privacy Policy - What may the AEE do with my information?
  • About - What version of the AEE am I using?
