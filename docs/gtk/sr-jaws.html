Step by Step - Install the AEE Configuration File for JAWS

The AEE Configuration File aee.jcf contains configuration directives for JAWS that can be used to apply recommended settings for the AEE. You can use the following steps to install these settings as a Domain Settings file in the JAWS Settings Folder, so they will be used for the AEE web site without affecting configuration settings for other applications.

First, use these steps to find the location of the JAWS Settings Folder on your machine using the JAWS Script Manager, and copy its location to the clipboard.

  • Type JAWS + 0 to open the JAWS Script Manager
  • Type Ctrl + O to open the Open User File dialog
  • Use Tab to navigate to the All Locations split button
  • Press Space to select the JAWS Settings Folder location
  • Press Ctrl + C to copy the location to the clipboard
  • Use Shift + Tab to navigate to the Cancel button
  • Press Space to dismiss the Open User File dialog

Next, use these steps to install the AEE Configuration File in the JAWS Settings Folder.

  • Use Tab to navigate to this link to the AEE Configuration File
  • Press Shift + F10 to activate the link context menu
  • Arrow down to select Save Link As and press Enter
  • Use Tab to navigate to the All Locations split button
  • Press Space to select the Download Folder path
  • Press Ctrl + V to paste the path from the clipboard
  • Press Enter to open the JAWS Settings Folder
  • Use Shift + Tab to navigate to the Save button
  • Press Enter to save the AEE configuration file

These steps need to be done only once, for each web domain used to access the AEE.
