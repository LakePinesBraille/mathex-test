Getting To Know the Equalize Editor

General Overview - What can the EE do for me?

The Equalize Editor (EE) is web application that creates Braille text and math documents.

The literary text in the document may be created by typing plain text on a computer keyboard, or uncontracted or contracted Braille on a braille keyboard.

The math formulas in the document may be created using the visual math editor on a computer keyboard, or using Nemeth braille symbols on a braille keyboard.

You do not need to use a screen reader to enter braille or to have braille generated for you. Visit the Home Row Braille topic to learn how. A screen reader is needed to collect braille input and to produce braille output using a refreshable braille keyboard if you have one.

The ability to create text and math documents using either a computer keyboard or a braille keyboard enables both sighted and braille users to create, save, edit, print, proofread, and share documents with others using accessible tools and file formats.
