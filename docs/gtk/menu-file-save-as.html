[File] Save As - How do I save a document to a new file?

Store a document to a new file in the local file system.

This menu operation stores the document using HTML and Content MathML to allow the Open operation to retrieve the text and math portions of the document.

You will be prompted to supply the name of the new file.

Type Alt Shift F, A to store a document a new file.

This operation modifies the file in the file system.
