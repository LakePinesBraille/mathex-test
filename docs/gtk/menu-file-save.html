Getting To Know the Equalize Editor

[File] Save - How do I save an existing document?

Store a document to the file last used to access it.

This menu operation stores the document using HTML and Content MathML to allow the Open operation to retrieve the text and math portions of the document.

You will not be prompted to supply the name of the file, unless it has not been saved before.

Type Alt Shift F, S to store a document to the file last used to access it.

This operation modifies the file in the file system.
