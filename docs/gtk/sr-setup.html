Getting To Know the Equalize Editor

Screen Readers - How do I get the screen reader to work?

The EE is designed to work with a wide range of screen readers, using as few custom configuration settings as possible. Some settings are required for the EE to work properly, while others may be set to any setting and the EE will function as designed.

You do not need to use a screen reader to enter braille or to have braille generated for you. Visit the Home Row Braille topic to learn how. A screen reader is needed to collect braille input and to produce braille output using a refreshable braille keyboard if you have one.

Use the links below to learn how to set up the screen reader you use to work with the EE.

If your screen reader is not listed below, visit the Other Screen Readers topic for general information on settings required for the EE to function properly.
