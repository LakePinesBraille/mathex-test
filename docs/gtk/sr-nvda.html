Getting To Know the Equalize Editor

NVDA Setup - How do I set up the EE configuration profile?

Summary

Download the file ee.ini into your NVDA User Configuration Folder, and make sure the name of the file is ee.ini.

Add the file gestures.ini to your NVDA gestures.ini file.

Use the keyboard gesture NVDA + E to activate and deactivate the ee configuration profile.

Step by Step - Install the EE Configuration File for NVDA

The EE Configuration File ee.ini contains configuration directives for NVDA that can be used to apply recommended settings for the EE. You can use the following steps to install these settings as a Configuration Profile in the NVDA Settings Folder, so they will be used for the ee configuration profile without affecting configuration settings for other configuration profiles.

First, use these steps to find the location of the User Configuration Folder on your machine, and copy its location to the clipboard.

  • Use the Windows key to open the Windows menu
  • Use Tab to navigate to the All Apps button and press Enter
  • Press N and arrow down to select the NVDA folder and press Enter
  • Arrow down to select Explore NVDA user configuration directory and press Enter
  • Use Tab to navigate to the All Locations split button
  • Press Space to select the NVDA User Configuration folder location
  • Press Ctrl + C to copy the location to the clipboard
  • Press Alt + F4 to dismiss the explorer window

Next, use these steps to install the EE Configuration File in the User Configuration Folder.

  • Use Tab to navigate to this link to the EE Configuration File
  • Press Shift + F10 to activate the link context menu
  • Arrow down to select Save Link As and press Enter
  • Use Tab to navigate to the All Locations split button
  • Press Space to select the Download Folder path
  • Press Ctrl + V to paste the path from the clipboard
  • Type \profiles to finish the path to the Configuration Profiles Folder
  • Press Enter to open the Configuration Profiles Folder
  • Use Shift + Tab to navigate to the Save button
  • Press Enter to save the EE configuration file

These steps need to be done only once.

The configuration settings included in this file are explained in the following sections, so you can choose those settings that will be most helpful for your accessibility needs.

The EE Gestures File gestures.ini contains configuration directives that may be added to the NVDA gestures.ini file to assign the keyboard gesture NVDA + E to activate and deactivate the ee configuration profile. You can use the following steps to add the EE keyboard gesture.

First, use these steps to copy the configuration directives to the clipboard.

  • Use Tab to navigate to this link to the EE Gestures File
  • Press Enter to open the EE Gestures File in a new tab
  • Press NVDA + F2 and then Ctrl + A to select all text
  • Press NVDA + F2 and then Ctrl + C to copy all text
  • Press Ctrl + F4 to dismiss the EE Gestures File

Next, use these steps to add the EE Gestures File to the NVDA gestures.ini.

  • Use the Windows key to open the Windows menu
  • Use Tab to navigate to the All Apps button and press Enter
  • Press N and arrow down to select the NVDA folder and press Enter
  • Arrow down to select Explore NVDA user configuration directory and press Enter
  • Press G to select the gestures.ini file
  • Press Enter to open the file in Notepad
  • Press Ctrl + V to paste the new EE gesture
  • Combine any duplicate configuration sections
  • Press Ctrl + S to save the updated gestures.ini file
  • Press Alt + F4 to close Notepad

These steps need to be done only once.
