Getting To Know the Accessible Equation Editor

Keyboard Input - How do I enter print and braille symbols?

The AEE supports keyboard input from a standard computer keyboard or from a refreshable braille keyboard. The AEE provides three major keyboard input modes to enter print and braille symbols using these keyboards.

You do not need a braille keyboard to enter braille or to have braille generated for you. Visit the Home Row Braille topic to learn how.

Use the links below to learn which keyboard input mode will work best for you, how to select a default keyboard input mode, and how to switch from one keyboard input mode to another.

  • Computer Keyboard - How do I enter print symbols?

    Sighted users should start here to learn how to enter text and math print symbols using a computer keyboard.

  • Refreshable Braille - How do I enter braille symbols?

    Braille users should start here to learn how to enter text and math braille symbols using a refreshable braille device.

  • Home Row Braille - How do I enter braille on a computer keyboard?

    Those without a refreshable braille keyboard should start here to learn how to enter braille symbols using the home row keys of a computer keyboard.
