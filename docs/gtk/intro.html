Getting To Know the Equalize Editor

Tutorial

Welcome to the Equalize Editor (EE) for Braille Text and Math documents!

This tutorial provides a gentle introduction to the EE. Press Escape at any time to close the tutorial and return to the application. Select Tutorial from the Help Menu to reopen the tutorial.

Use the links below to explore each topic, and use Backspace to return to the topic you just left. Press Home at any time to return to this tutorial introduction from any other topic.

Be sure to visit the Initial Setup topic to prepare the EE to work with your screen reader and your electronic braille device.

Please see the Users Guide for more details on how to use the EE, including keyboard input, braille input, application menu items, and screen reader configuration information.
