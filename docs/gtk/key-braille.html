Refreshable Braille - How do I enter braille symbols?

Braille input mode allows events from a refreshable braille keyboard to be received as braille symbol input for text and math characters.

Braille input mode is typically used with a refreshable braille device using its braille keys to enter braille symbols that correspond to contracted braille text and Nemeth Braille math.

The braille keyboard determines how to enter Ctrl and Alt characters using the braille keys.

You can type the sequence == to enter a control character, to support braille keyboards that may not provide a way to type control characters.

Typing dots-123456 on a braille keyboard enters the = key, so you can also type dots-123456 dots-123456 as a way to type control characters.

Since Braille input mode is designed for entering Computer Braille using a braille keyboard, sighted users who are proficient at Computer Braille may find it a useful way to simulate braille input using a computer keyboard.

A more direct way to input braille without a braille keyboard is to use Home Row Braille.

Type Ctrl r or ==r or dots-123456 dots-123456 dots-1235 to switch to Braille input mode.

The current keyboard input mode may be changed on the Settings Page, and persists when you revisit the EE web application.
