Drive Menu - How do I share my work?

The Drive Menu provides tools to load and store your documents to the Google Drive shared file system so others may access them.

Type Alt Shift D to navigate to the Drive Menu.

Use the links below to explore each item on the Drive Menu.

  • Open - How do I open an existing shared document?
  • Save - How do I save an existing shared document?
  • Save As - How do I save a document to a new shared document?
  • Save Replace - How do I save a document to replace an existing shared document?
